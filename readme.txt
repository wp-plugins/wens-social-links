== Description ==

Plugin that can be used to link to 20 social networking sites .

= Usage Instructions =

1. Activate the plugin from the WordPress Dashboard

2. Go to WEN Social Links under Settings menu 

3. Fill in the desired fields and we're good to go

4. You can find instruction to use plugin in Setting page


== Installation ==

1. Download the plugin from the link above

2. Extract the file contents 

3. Upload the directory to your WordPress plugins' directory ( /wp-content/plugins/ )

4. Activate the plugin from the WordPress Dashboard

5. Go to WEN Social Links under Settings menu

6. Fill in the desired fields and we're good to go

== Frequently Asked Questions ==

= How can I change style of this plugin ? =

You can over-write the style of this plugin. You can use Custom Style field in plugin's setting page.

= Can I change the icons ? =

Yes. Just can replace the icon style from your CSS code .

= Can I display either of the Social Statistics ? =

Yes. Simply enter the data for only those items that you would like to display .

= Can I enable or disable tooltips? =

Yes. You can enable or disable it in plugin's setting page.
