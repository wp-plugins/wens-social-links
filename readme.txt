== Description ==

A simple plugin that can be used to link to 21 social networking sites .

= Usage Instructions =

You can use this plugin with 2 methods.

= Method 1: Using as Widget =

1. Activate the plugin from the WordPress Dashboard

2. Go to widgets and drag the WEN: Get Social Widget where you want the link (s) to appear

3. Fill in the desired fields and we're good to go

= Method 2: Using Shortcodes =

1. `[socialiconsall title="your-title" tooltip="1" rss="rss-link" facebook="facebook-link" twitter="twitter-link" gplus="gplus-link" linkedin="linkedin-link" pinterest="pinterest-link" digg="digg-link" myspace="myspace-link" tumblr="tumblr-link" flickr="flickr-link" reddit="reddit-link" dribbble="dribbble-link" blogger="blogger-link" yahoo="yahoo-link" aim="aim-link" skype="skype-link" paypal="paypal-link" youtube="youtube-link" vimeo="vimeo-link" dailymotion="dailymotion-link" netflix="netflix-link"]`

2. `[socialiconsall rss="rss-link" facebook="facebook-link" twitter="twitter-link" gplus="gplus-link" linkedin="linkedin-link" pinterest="pinterest-link" digg="digg-link" myspace="myspace-link" tumblr="tumblr-link" flickr="flickr-link" reddit="reddit-link" dribbble="dribbble-link" blogger="blogger-link" yahoo="yahoo-link" aim="aim-link" skype="skype-link" paypal="paypal-link" youtube="youtube-link" vimeo="vimeo-link" dailymotion="dailymotion-link" netflix="netflix-link"]`

== Installation ==

1. Download the plugin from the link above

2. Extract the file contents 

3. Upload the directory to your WordPress plugins' directory ( /wp-content/plugins/ )

4. Activate the plugin from the WordPress Dashboard

5. Go to widgets and drag the WEN: Get Social Widget where you want the link (s) to appear

6. Fill in the desired fields and we're good to go

== Frequently Asked Questions ==

= How can I change style of this plugin ? =

You can over-write the style of this plugin from your template style file .

= Can I change the icons ? =

Yes. Just can replace the icon style from your CSS code .

= Can I display either of the Social Statistics ? =

Yes. Simply enter the data for only those items that you would like to display .

= Can I enable or disable tooltips? =

Yes. You can enable or disable it from widget backend if you are using widget. If you are using shordcode use shordcode as specified in the description section.
